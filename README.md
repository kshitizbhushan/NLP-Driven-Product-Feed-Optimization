# NLP-Driven Dynamic Product Feed Optimization

## Project Overview

This project implements a semantic matching engine that connects trending social media keywords with products from an e-commerce catalog using Natural Language Processing (NLP).

Instead of relying on traditional keyword matching, the system uses Sentence Transformers to understand semantic meaning and FAISS for efficient similarity search.

The final output identifies the Top-3 matching trends for every product and generates optimized product titles.

---

## Features

- Exploratory Data Analysis (EDA)
- Text Preprocessing
- Semantic Embedding Generation
- Sentence Transformers
- FAISS Similarity Search
- Cosine Similarity
- Optimized Product Title Generation
- CSV Export
